# cs81-module3a-github
This was started as part of CS81 coursework requirement at SMC, and this description will probably evolve.

## greet
Greets a user by name

## square
Squares a number by multiplying it by itself

## greet(who)
Using concatenation, greet (who) and then close by saying "Bye"

## power
A more complicated way to resolve exponents by establishing a base and an exponent, and multiplying via loop until the exponent is 0, i.e. it is multipled x number of times, then by itself

## arrow (h)
A less verbose way of dividing by a 3, with possible values being 0, 1, or 2; could be used as a succinct way of sequencing something
